ADMIN DASHBOARD

Project Description

This web app is developed using a combination of HTML, CSS, and React technologies. It leverages an Application Programming Interface (API) to efficiently retrieve random recipes from a vast collection of over 250 culinary creations. Within this culinary exploration framework, users gain the ability to initiate a gastronomic journey, revealing a diverse range of delightful recipes to stimulate their palate. Moreover, this interactive platform provides users with a spectrum of engagement features, such as:


  • Watch Tutorial: Users can watch cooking video tutorials for selected recipes on Youtube with the click of a button
  • Read Blog: Users can read a blogs of the recipes
  • Download PDF: Users can download printable PDF versions of their favorite recipes
